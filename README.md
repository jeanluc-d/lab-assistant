## Setup

To get the node modules installed, run:

```shell
npm install
```

## Running the app

To get the app up and running (default port 8080), run:

```shell
npm start
```

## Running the app in dev mode

To get the app up and running, run:

```shell
npm run dev
```

## Building the app

To build the app, run:

```shell
npm run build
```
